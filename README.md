# Personal CV - GitHub Pages

This is my personal CV/resume website hosted on GitHub Pages.

## Setup

1. Edit `index.html` to add your personal information:
   - Replace "Your Name" with your actual name
   - Update contact information (email, phone, LinkedIn, GitHub)
   - Fill in your experience, education, skills, and projects

2. Customize `style.css` to match your preferred color scheme and styling

3. Enable GitHub Pages:
   - Go to your repository settings
   - Navigate to "Pages" in the left sidebar
   - Select the main branch as the source
   - Your site will be available at `https://franpt0510.github.io`

## Features

- Clean, professional design
- Responsive layout (works on mobile and desktop)
- Easy to customize
- No external dependencies

## Customization

Feel free to modify the HTML structure and CSS styling to match your personal brand and preferences.
